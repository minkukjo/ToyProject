## Spring Boot + Vue를 활용한 ToDoApplication ToyProject

본 프로젝트는 Spring Boot와 Vue 학습을 위한 토이 프로젝트입니다.

## 앞으로 추가해야할 사항

1. 현재 DB가 MongoDB인데 ElasticSearch로 변경 후 테스트
2. BackEnd에 Delete 기능이 구현되어있지 않음, 구현 후 Front에 버튼을 만들어 테스트
